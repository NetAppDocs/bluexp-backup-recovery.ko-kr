= Kubernetes 워크로드 관리 개요
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
BlueXP backup and recovery 에서 쿠버네티스 워크로드를 관리하면 쿠버네티스 클러스터와 애플리케이션을 한 곳에서 관리하고 보호할 수 있습니다. 쿠버네티스 클러스터에서 호스팅되는 리소스와 애플리케이션을 관리할 수 있으며, 쿠버네티스 워크로드의 보호 설정도 관리할 수 있습니다.


NOTE: 이 문서는 기술 미리보기로 제공됩니다. 미리보기 기간 동안 Kubernetes 기능은 프로덕션 워크로드에 권장되지 않습니다. 이 사전 공개 오퍼링과 함께 NetApp은 일반 출시 이전에 서비스 세부 정보, 콘텐츠 및 일정을 수정할 권한을 보유합니다.
